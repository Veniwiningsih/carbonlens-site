What CarbonLens Does
📊

ESG Report Parsing

Automatically extract and structure data from PDF ESG reports, sustainability disclosures, and annual reports using advanced NLP.

🔍

Greenwashing Detection

AI-powered analysis that identifies misleading environmental claims by cross-referencing stated goals with actual performance data.

📋

CSRD Compliance

Automated gap analysis against Corporate Sustainability Reporting Directive requirements. Maps disclosures to ESRS standards.

🌿

EU Taxonomy

Determine alignment with EU Taxonomy Regulation. Assess substantial contribution, do no significant harm, and minimum safeguards.

🌡️

Paris Agreement

Validate corporate climate targets against Paris Agreement pathways. Check 1.5°C and 2°C alignment with SBTi methodologies.

⚖️

Peer Benchmarking

Compare ESG performance across industry peers. Identify leaders, laggards, and outliers with multi-dimensional scoring.

How It Works
10-Step Reasoning Chain
01

Document Ingestion

Parse PDF, HTML, or XBRL sustainability reports into structured text segments.

02

Entity Extraction

Identify companies, metrics, targets, dates, and regulatory references.

03

Metric Normalization

Convert all emissions data to standardized units (tCO2e) with consistent boundaries.

04

Claim Verification

Cross-reference stated claims against actual reported data points.

05

Baseline Comparison

Compare current performance against historical baselines and reduction trajectories.

06

Regulatory Mapping

Map disclosures to CSRD, EU Taxonomy, TCFD, and GRI framework requirements.

07

Greenwashing Scan

Detect vague language, cherry-picked metrics, and misaligned offset claims.

08

Peer Benchmarking

Rank performance against industry peers using multi-factor scoring.

09

Risk Assessment

Quantify transition risk, physical risk, and regulatory compliance gaps.

10

Report Synthesis

Generate structured JSON output with scores, flags, and actionable insights.

Developer API
Get Started in 4 Lines
analyze.py
from carbonlens import ESGAnalyzer # Initialize with multi-agent reasoning analyzer = ESGAnalyzer(reasoning="step-by-step", model="hermes-3") # Analyze any ESG report result = analyzer.analyze("reports/apple_esg_2024.pdf") # Access structured results print(result.esg_score) # 87 print(result.greenwash_risk) # "low" print(result.scope3_emissions) # 22_100_000 tCO2e print(result.reasoning_steps) # 10 detailed steps
